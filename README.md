# Citrix_QA_Automation


## Tools
   * Java
   * Selenium WebDriver
   * ChromeDriver
   * Maven

## To execute via command line

    ```
        mvn clean install
    ```

## Tested with
   * Chrome
   * Firefox

## Test Config

Please use config.js to change the properties for test executions

    ```
    baseUrl https://global.gotowebinar.com
    userName <provideUserName>
    password <password>
    driver Chrome (| Firefox)
    ```
